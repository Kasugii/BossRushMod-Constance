# Boss Rush Mod — Constance

A **BepInEx** mod for the game **Constance** that adds a Boss Rush mode with a global online leaderboard.

**Press F1** in-game to open the Boss Rush menu.
**Press F3** to toggle the corrupted skin permanently.

---

## Features

- **Boss Rush Mode** — GUI selector to choose which bosses to fight and in what order
- **3 run sizes** — Back-to-back boss runs without healing : 5 bosses, 10 bosses, or all 16 (random order)
- **Curated lists** — 3 handcrafted lists per size (5-boss and 10-boss), each covering all available bosses
- **Live timer** — On-screen HUD showing elapsed time, current boss, and next boss
- **Online leaderboard** — Submit and view scores by category
- **Brian difficulty selector** — Choose Easy / Normal / Hard / Extreme before each Brian fight

---

## Requirements

- [BepInEx 5.4.x](https://github.com/BepInEx/BepInEx/releases) for Constance
- The game **Constance**

---

## Installation

1. Install BepInEx into your Constance game folder
2. Download the latest release `.dll`
3. Copy it into `BepInEx/plugins/`
4. Launch the game — the Boss Rush shrine will appear in the hub area
